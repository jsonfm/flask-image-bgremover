### 🌌 BgRemover

An image background remover made with Python/Flask and HTML/JS.

### ⚙️ Technologies

-   Flask
-   Pillow
-   Rembg
-   HTML/CSS/JS
-   Alpine.js

### 📦 Installation

```
pip install -r requirements.txt
```

### ⚡️ Development

```
flask --app main run --debug
```
